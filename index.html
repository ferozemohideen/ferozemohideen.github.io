Projects

What I've done

Peak detection algorithm

A MATLAB project completed at AxoSim that analyzes the peaks of an electrophysiological wave generated by nerve stimulation, cancels out the noise and the stimulus artifact, then intelligently chooses the latency of the first peak. This data can then be used to calculate the nerve conduction velocity of the channel under the effects of drugs of differing concentration. This collection of scripts turned a 10-hour-long process of manually subjectively choosing the right peak for each file into a 10-minute-long procedure that can be relied upon for objective results.

Technologies:
MATLAB, Python
